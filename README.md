## Hướng dẫn cài đặt:
* Nếu chưa cài đặt npm, cài đặt npm theo link (https://nodejs.org/en/download/)
* Chạy `npm run build` để build project.
* Chạy `npm run start` để start project.
* Project được khởi chạy trên (http://localhost:8080/)
